UPGRADE INSTRUCTIONS
----------------------------------------------------------------
When you receive updated files:

  1. Check this manifest — note which files changed
  2. Copy new files to /home/pi/synthos/ (no version suffixes)
  3. Add new .env keys listed above
  4. Run: python3 patch.py --status
  5. Or use patch.py for safe automatic updates:
     python3 patch.py --dir /path/to/update/folder/
  6. Update this manifest with the new version and date

NEVER manually edit signals.db, .env, or credentials.json
NEVER overwrite these files during an update — they are your
trade history, API keys, and authentication credentials.
