Free / Public API Source List

Working list for staged expansion of your system. This is organized so you can unlock sources over time instead of integrating everything at once. “Core” means high practical value early. “Expansion” means useful after the pipeline is stable. “Later” means high-noise or heavier filtering work.

Note: Some sources are fully open, while others require a free API key or account registration. Always check the source’s current rate limits and terms before integrating.

Suggested unlock order

  • Phase 1: SEC EDGAR, Congress.gov, Federal Register, FRED, USAspending
  • Phase 2: openFDA, Treasury Fiscal Data, BLS, Census, GovInfo
  • Phase 3: OpenCorporates, NIH, ClinicalTrials.gov, USPTO, Companies House
  • Phase 4: GDELT and broader news APIs after your dedupe / credibility layer is stable

Practical caution

  • Do not unlock high-noise news APIs before your storage, dedupe, and rate-limit controls are in place.
  • Prefer official agency APIs and public filings first.
  • Treat this list as an expansion menu, not an integration checklist.
